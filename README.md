# homebrew-termdown

Homebrew tap for [termdown](https://github.com/dsaad68/termdown), a minimal
terminal Markdown browser written in pure Swift.

## Install

```sh
brew install dsaad68/termdown/termdown
```

Works on macOS and Linux. Upgrade later with `brew upgrade termdown`.

## Notes

`Formula/termdown.rb` is generated automatically by termdown's release workflow
on each `v*` tag; do not edit it by hand.
