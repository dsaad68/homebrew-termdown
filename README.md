# homebrew-termdown

Homebrew tap for [termdown](https://github.com/dsaad68/termdown) — a terminal
Markdown browser written in pure Swift.

```sh
brew tap dsaad68/termdown
brew install dsaad68/termdown/termdown
```

> While the source repository is private, tapping and installing require a
> `HOMEBREW_GITHUB_API_TOKEN` (a GitHub token with `repo` scope).

`Formula/termdown.rb` is **generated automatically** by termdown's release
workflow on each `v*` tag — do not edit it by hand. The initial commit carries
placeholder checksums until the first release populates them.
